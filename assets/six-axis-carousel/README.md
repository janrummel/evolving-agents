# Six-Axis Carousel · LinkedIn Post Asset

A 6-slide carousel introducing the Six-Axis-Maturity-Model for AI agents, with the Reflection-Gap of industry frameworks as concrete hook. Audience: AI/ML research-adjacent practitioners.

## Files

- `slide-{1-6}-*.html` — six static HTML slides at 1200×627
- `_base-template.html` — shared CSS template
- `render-all.sh` — Headless-Chrome → PNG renderer
- `post-text.md` — LinkedIn caption + hashtags + posting notes
- `slide-{1-6}.png` — generated by render-all.sh (gitignored)

## How to render

```bash
./render-all.sh
```

Produces `slide-{1-6}.png` in this folder. Requires `/Applications/Google Chrome.app/`.

## How to post

1. Render PNGs: `./render-all.sh`
2. Open LinkedIn → "Create post" → Image carousel
3. Upload `slide-1.png` through `slide-6.png` in numerical order
4. Paste caption from `post-text.md` (Caption section) into the post body
5. Add hashtags
6. Schedule or publish

## Source

Substance is derived from a private cross-model maturity-framework analysis project. The carousel only references publicly available sources (arXiv papers, public framework documentation, Nous Research's open-source Hermes Agent). No private methodology, internal repos, or non-public empirics are surfaced.
