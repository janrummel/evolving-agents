# Evolving Agents

**From Nowak's Evolvability Equations to AI Agent Architectures**

A living research collection and practical design principles at the intersection of evolutionary dynamics, self-evolving agents, and multi-agent systems.

## What This Is

Two things in one repo:

1. **Research** — A curated, growing knowledge base connecting Martin Nowak's mathematical framework for evolution with modern AI agent systems. Papers, syntheses, open questions.
2. **Principles** — Actionable design principles derived from evolutionary theory, applicable to real agent architectures.

## Why This Matters

Current AI agent systems are fundamentally static. Nowak's work on evolutionary dynamics offers a mathematical lens to understand *when* and *how* agent systems can evolve — and what breaks when they try.

Key insight: There's a formal isomorphism between biological evolution (replication, mutation, selection) and agent system improvement (workflow reuse, prompt variation, evaluation).

## Core Topics

- **Nowak's Originator Equation** — The math of prelife-to-life transitions, mapped to agent systems
- **Evolvability ES & Quality-Diversity** — Optimizing for adaptability, not just performance
- **Self-Evolving Agents** — Parameter, prompt, workflow, tool, and memory evolution
- **Multi-Agent Collaboration** — Attribution problems, failure modes, cooperation metrics
- **Error Threshold** — The formal limit of how much you can change before evaluation breaks down
